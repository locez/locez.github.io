# 回顾 2019
---

2019 年对我来说，应该是最近几年变动比较大的一年了，最主要是从学校毕业了，但是我
从来就不想毕业。不得不说这种角色的转变，我至今还不想承认我毕业了的事实，因为还是
读书爽啊。还是从今年几个事件，时间来回顾今年发生了什么吧。


<a id="orga560583"></a>

## 毕业
---

从大四下开学就忙起了毕业论文的事情，为了多学一点东西，当时选题的时候就选了自己完
全不会的内容，大数据方面的，不知道为啥，我只是想趁着这个机会多学点东西，毕竟当时
很多人都在炒这个概念。但是真的当自己有接触有了解以后，才发现这个东西并没有宣传的
那么玄乎其玄，也没有那么厉害。从 0 接触到最后完成一个大数据的流式处理，应该也就
花了 10 多天吧，记得自己每天都在深夜躺着查资料，查完第二天就把代码写出来。js 也
是这个时候学的，很皮毛，因为我只需要最后将结果展示出来即可。起初是用了 java 写了
一天，但是真的不是很喜欢 java，然后就换成了 scala，感觉挺好的。后来学习 rust 的
时候，因为有 scala 的基础，对 Option 这些东西理解起来很方便，这大概就是接触了一
种语言的某种特性，其它都会发散开来吧。论文格式后来调了挺长一段时间的，归根结底还
是因为这个大学居然连个 LaTeX 都没有，本来我是打算趁着写论文把 LaTeX 也学了，这样
以后需要排版的时候可以用上。最后就是稀里糊涂毕业了，但是跟别人说我还没毕业。


<a id="orgf0b768d"></a>

## 开始工作
---

记得经理是在 4 月份问我入职么？我当时马上拒绝了（用的是论文还没写好，其实早就写
好了），因为我稍微一思索，我以后还有几十年要工作，我为什么要这么早去工作，我根本
不喜欢工作，其实这就是我对工作的态度。我只是想一直当个学生，做自己喜欢的事就可以
了，要说为什么工作，可能只是因为 2020 年要被消灭了，被迫去维持生计而已。

入职网易，我不知道这是好事还是坏事，有的人说我不知满足，网易这么好你还嫌弃什么?
其实我关心的都不是这些，而是我是不是在这里做着我喜欢做的事情。还记得去年实习的时
候，做完数据库研究，我就马上提了离职申请，走离职流程了。只不过后来被导师推荐到现在的
组了，又写了 2 个月 go，感觉稍微还可以。我当时明确表示我不想做运维，运维不是我要做的活，所以就被推过去当开发了。其
实入职这几个月，我也发现了，这个开发也不是我想象中的开发，我明明当时是想进来做一
些 go 的开发的，因为我从来没说我会 python，我只会 ruby，然后强行把我变成了一个
python 开发，，这仍然不是我想做的工作，所以我根本没什么动力学这个东西。
我不觉得我很厉害，我一直都觉得我很菜，但是我想学一些东西，一些真正有用的东西，而
不是说一些看起来就很简单的，然后把这些复杂化，强行变得看起来很困难，这样做就一定
是正确的方法论。

抛开技术层面的诉求没有达到我的预期之外，其它都还不错，人际关系什么的也比较和谐，
没有传闻中的职场那么恐怖，大佬们说话都很好听，也很好相处。重点是不怎么加班，所以
抛开技术层面我还是比较喜欢在这里的。

## 阅读学习
---

我博客有个 reading 的页面，那个页面是我以前还喜欢读书的时候会记录下来读了什么，
已经很久没有更新了，因为已经几年没有怎么读书了。但是今年好不容易终于又把这个习惯捡起来了，读书真的是一件很幸
福，但是又因为很多日常的杂事，而无法持久的坚持下去的一件事。不过今年还是读了不少书的，
在微信读书上面白嫖，实体书也有（偏少）。让我印象比较深刻的还是 《小王子》 与
《月亮与六便士》。

在 《月亮与六便士》中，简直就是对梦想与现实的灵魂拷问，斯特里克兰的奋不顾身才实
现了他最终的梦想，我非常敬佩他的这种勇气，舍弃了一切，只为了那个梦，当然他许多不
道德的事，我还是不能认同。可是在追逐梦想的过程中，我们能轻易的放弃我们现在所拥有
的东西么？我问过我自己，现在放弃这个工作，再回去考虑读研，我有那么大的决心么？能
一定考成功么？我不敢，因为我不觉得我可以考上，也不知道考研的那种读书是不是真的是
自己喜欢的方向。

而在《小王子》中，可能我读的是一本哲学书籍吧，我看到了许多道理，得到了许多说不出
的触动人，同时也感觉产生了共鸣，大人们总是觉得他们的见解是对的，其实他们什么都不懂，
总是要我们小孩子来解释。其中里面还有一句：“人总是对自己所处的地方不满意”，这大概
挺对应我的场景的。

“就像那朵花，如果你爱上一朵生长在某个星球上的花，当你抬头望着夜空时，你会感受到
很甜蜜。仿佛所有的星星都开满了鲜花。”

还有在读的就是一些技术书籍，不多基本是在研究 rust 吧，rust 是我认为即使不用这个
来写程序也值得学习的一门语言之一，因为他没有屏蔽掉那些细节，需要注意的点。要学会
rust 必须懂得内存的分配，是在堆还是在栈上，这些是更上层的语言根本不会关注以及告
诉你的。我只是通过一门语言去学习相应底层的一些知识罢了。而最重要的一点就是 rust
像个大锅一样，揉杂了很多东西进去，如果想通过一门语言学习到很多语言的一些知识，感
觉是一个不错的选择。

## 生活
---

生活除了从灵魂时差变成正常作息，然后每天上班，周末休息之外，似乎没有太大的变
化。主要变化还是在家里，不知道为什么，曾经从不催我找女朋友的父母，现在开始每个电
话都要问我出去玩没有，有没有认识女孩子，有没有去找女朋友谈恋爱什么的，我都开始反
感有点腻了。不知道为何，毕业了难道这些就是上一辈觉得最优先的事情么？倒也不是说不
能催，就是每个电话都要提一下就变味了。差点我就在电话里说出再逼我就要女装了，列表
一堆人还想看我女装呢，可恶。打点钱吧，都要被消灭了，没钱女装啊。

# 2020
---

写着写着就从 2019 写到 2020 年了，2019 已经过去了，说再多的也只能是感触。
SteamedFish 大佬说：“总结：2020 年一定要开始进行 2019 年发誓要做的原定于推迟到
2018 年完成的 2017 年度计划”。这种情况其实我存在很久了，大学散漫了好几年，毫无规
划，不知道在干嘛。所以那几年书也没怎么读，笔记也没有，博客也没有，代码产出更没有。
2020 年这种状况我还是希望可以改善一下的，毕竟如果一件事都没完成，回顾也好，总结
也好制定计划也好，都没有意义，曾经制定的计划不够公开透明，所以也缺乏动力，但是今
年我选择直接在博客放出来，自己能看见，有一个地方记录，也算是一个 flag，其他人看见
也能监督我。
