### POEM AND CODE

---

早在痴迷代码之前就很喜欢诗词，特别喜欢诗词描述的那种意境美。只因生活的忙碌让我逐渐无力再去体会，但是我还是想将曾经的摘录在此，也不仅仅包含诗词，还有其它东西。

```
一行代码，一行诗。
        Locez 2017.11.08
```

```
一行代码一行诗，不问前程行好事。
此路无期须独行，染尽霜雪莫回头。
        Locez 2019.09.23
```

```
我们总会在月亮上相遇的
```

```
雨落黄昏花易落，碧海青天夜夜心。
```

```
《采桑子·时光只解催人老》
    晏殊

时光只解催人老
不信多情
长恨离亭
泪滴春衫酒易醒
梧桐昨夜西风急
淡月胧明
好梦频惊
何处高楼雁一片
```

```
走的走，来的来，你面前总有几张板凳，板凳总没空过。
        《自在独行·朋友》
```

```
不管前方的路有多苦，只要走的方向正确，不管多么崎岖不平，都比站在原地更接近幸福。
```

```
人生如逆旅，我亦是行人。
```

```
少壮不努力，老大徒伤悲。莫等闲、白了少年头，空悲切。
```

```
彼时当年少，莫负好时光。
```

```
即便这个世界处处充满恶意，也还是守住角落那最后的善意。
```

```
唯有再见方为人生。
```

```
情用赏为美，事昧竟谁辩。
```

```
棠梨花映白杨树，尽是死生别离处
```

```
梅似雪、柳如丝、试听别语慰相思
```

```
行走在莫比乌斯环之上
```

```
白雪却嫌春色晚，故穿庭树作飞花。
```

```
人生如寄，多忧何为？
```

```
昨夜蝶衣刚入梦，珍重，东风要到送春时。
```

```
无人知此意，歌罢满帘风
```

```
海棠未雨，梨花先雪，一半春休。
```

```
雪粉华，舞梨花，再不见烟村四五家。
```

```
海棠红近绿阑干，才卷珠帘缺又、晚风寒。
```

```
时有落花至，远随流水香。
```

```
杨柳风柔、海棠月淡、独自依阑时。
```

```
雨打梨花深闭门，忘了青春，误了青春。
```

```
菩提本无树，明镜亦非台。
本来无一物，何处惹尘埃。
```

```
世间无限丹青手，一片伤心画不成。
```
